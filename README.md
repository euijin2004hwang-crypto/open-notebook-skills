# Open Notebook Skills

**Open Notebook** ([lfnovo/open-notebook](https://github.com/lfnovo/open-notebook)) Hermes Agent skill collection.

Open Notebook is an open-source AI research assistant and knowledge management tool.

## Skills

### open-notebook (English)
Guide for AI agents to use the Open Notebook REST API.
- Notebook search / RAG-based Q&A / Source and note management
- Podcast generation / Embedding rebuild / Model configuration
- Hermes MCP integration

### open-notebook-kr-guide (Korean)
Detailed Korean guide for Open Notebook.
- 3-step RAG chat workflow
- Curl examples with proxy bypass
